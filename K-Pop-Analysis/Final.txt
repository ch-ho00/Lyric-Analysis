This repository contains three main files.
    1. word2vec_main.py
    2. main_kiwi.py
    3. main_koala.py

Functionality of the word2vec_main.py file is to:
    1. Find the similiarity between two song id.
    2. Find the 3 top closest songs

Functionality of main_kiwi.py is to:
    1. Generate kiwi BOW of lyrics and save cleaned result
    2. Generate LDA model

Functionality of main_koala.py is to:
    1. Generate koala BOW of lyrics and save cleaned result
    2. Generate LDA model
